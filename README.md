# Credit-Card-Dataset-for-Clustering

Esta aplicación web interactiva utiliza técnicas de Machine Learning no supervisado para analizar el comportamiento financiero de usuarios de tarjetas de crédito. Basada en un dataset de +8900 registros, la herramienta permite ingresar métricas financieras de un nuevo cliente y determinar en tiempo real a qué segmento de mercado pertenece.

Características Principales:

Modelo de Clustering: Implementación del algoritmo K-Means para agrupar clientes en 4 perfiles estratégicos basados en patrones de compra, pagos y uso de crédito.

Reducción de Dimensionalidad: Uso de PCA (Análisis de Componentes Principales) para transformar 17 variables complejas en una visualización 2D comprensible.

Visualización Interactiva: Gráfico dinámico con Plotly que mapea la posición del nuevo cliente frente a los clústeres históricos.

Despliegue Cloud: Aplicación accesible globalmente alojada en Streamlit Cloud.

Tecnologías:

Lenguaje: Python 3.10

ML & Data: Scikit-learn, Pandas, NumPy.

Visualización: Plotly Express.

Frontend/Deploy: Streamlit.
